docker/mysqlError

Jan 21, 2020Hits

Client does not support authentication protocol requested by server; consider upgrading MySQL client

解决方法：

mysql -u root -p

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'Ll123456'

flush privileges;
